Modern beheer systeem voor stemlijsten en stemmen voor de Indie500.

== Description ==

Indie500 Manager is een moderne WordPress plugin voor het beheren van muziek stemlijsten en het verzamelen van stemmen. Perfect voor radio stations, muziek blogs, of iedereen die een muziek voting systeem nodig heeft.

**Features:**
* CSV upload voor stemlijsten
* Gebruiksvriendelijk stemformulier met zoekfunctie
* Real-time stemresultaten
* Archief functionaliteit per jaar
* Export functionaliteit voor resultaten
* Responsive design
* Anti-spam bescherming (IP-based)
* Admin dashboard met statistieken

**Shortcodes:**
* `[indie500_vote_form]` - Toon het stemformulier
* `[indie500_archive]` - Toon archief per jaar
* `[indie500_results]` - Toon huidige resultaten

== Installation ==

1. Upload de plugin bestanden naar `/wp-content/plugins/indie500-manager/`
2. Activeer de plugin via het 'Plugins' menu in WordPress
3. Ga naar 'Indie500' in het admin menu
4. Upload je eerste CSV bestand met titels
5. Gebruik de shortcodes op je pagina's
